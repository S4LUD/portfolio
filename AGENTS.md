# AGENTS.md

## Working Preferences

- Do not run a quick build check after every small UI change.
- Only run build or lint checks when explicitly requested, when making substantial changes, or when verification is actually needed.
- Do not auto-push to GitHub unless explicitly requested.
- When we are in a production-ready pass, every change must map to an item in `PRODUCTION_READY_CHECKLIST.md`.
- During a production-ready pass, review the checklist before making changes and update it as items are completed.
